如何下载nginx并且项目部署服务器

发表于 2019-08-25

前言

前言：由于最近完成了两个项目 需要将项目部署到服务器让大家能看到所以开始用起了nginx

1.Mac下安装homebrew

  • 1.1 将brew的install文件下载本地

  • 1.2 修改install文件的镜像源

    访达用command+shift+G找到brew_install文件进行修改）将

    改为

    我的install文件里没有大家说的CORE_TAP_REPO
    所以我在BREW_REPO下填加

  • 1.3 开始安装

  • 1.4.替换homebrew源(这一步没懂，但是没有影响我的操作)

2.vue项目打包

  • 2.1 配置打包文件

    配置文件一：build>>utils.js (修改publicPath:”../../“ , 这样写是处理打包后找不到静态文件的问题)

    配置文件二：config>>index.js(修改assetsPublicPath:’./‘ ,修改目的是为了解决js找不到的问题)

  • 2.2 控制台输出

  • 2.3 生成了一个index.html文件，可以直接打开访问

    3. 启动ngnix

    4.自定义网站和端口

  • 4.1 复制网站TanWeb文件夹物理路径，打开终端，准备编辑nginx的配置文件

    /Users/admin/Desktop/硅谷外卖2/dist

  • 4.2 打开终端进入

    进入nginx.conf页面后，按 “i” 键进入编辑状态，
    自定义端口，和配置本地网站TanWeb, 注意设置访问权限( user root owner; )，不然等会访问网站会出现403错误

    按esc键退出编辑状态，输入 :wq 保存退出nginx.conf页面

  • 4.3 重新启动nginx

    sudo nginx -s reload

  • 4.4 在浏览器输入localhost加自定义的端口，就能访问到配置好的网站了，

    http://localhost:5188/
